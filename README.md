# Long Term Support

To keep less work for me, I will try to maintain
4 versions of Minecraft for my mods, this 
includes (with reasons):

- 1.20.1   - I still play on this
- 1.21.1   - Next LTS version for modpacks
- latest   - Obviously
- latest-1 - aka n-1, to lengthen support.

I will NOT (back)port my mod to any version
unless there's enough demands for it. I also
reserve the right to drop support for old
version if they became unnecessary for me.

Making mods take a lot of my time and sanity,
and currently I have not be able to make any
incentive towards my works. So if you can,
donate to my Ko-Fi to keep me going.
